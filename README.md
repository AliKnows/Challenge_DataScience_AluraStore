# ***Proyecto de Ciencia de Datos de la Tienda AluraStore para el challenge de DataScience de AluraLatam***
## El proposito es realizar un analisis de los datos de diferentes tiendas para verificar cual de las cuatro tiendas es la mas y menos producttivas para tomar una decision
## El proyecto esta organizado en una serie de Notebooks, de indices en el cual es facil de navegar 
## Dentro del proyecto se logra ver una serie de graficos para un mayor entendimiento sobre el trabajo del analisis
